深圳友讯达科技股份有限公司关于重大经营活动中标的公告
本公司及董事会全体成员保证信息披露内容的真实、准确和完整，没有虚假记载、误导性陈述或重大遗漏。
广州供电局有限公司近期发布了《广州供电局 2018 年营销计量类物资框架招标中标公示》，深圳友讯达科技股份有限公司（以下简称“公司”）中标 3 个项目，现将具体事项公告如下：
法定代表人：甘霖
注册地址：广州市天河区天河南二路 2 号
经营范围：电力供应 ; 承装（修、试）电力设施 ; 送变电工程专业承包 ; 工程和技术研究和试验发展 ; 能源技术研究、技术开发服务 ; 电子、通信与自动控制技术研究、开发 ; 能源技术咨询服务 ; 工程和技术基础科学研究服务 ; 电力电子技术服务 ; 信息技术咨询服务 ; 电气防火技术检测服务 ; 工程技术咨询服务 ; 职业技能培训（不包括需要取得许可审批方可经营的职业技能培训项目） ; 电气设备零售 ; 电力输送设施安装工程服务 ; 建筑物电力系统安装 ; 电工器材零售 ; 企业自有资金投资 ; 投资管理服务。
是否存在关联关系：公司与广州供电局有限公司无关联关系
经广州供电局 2018 年营销计量类物资框架招标中标公示，公司获得三个项目中标候选人资格，分别为低压集抄系统，中标比例 40% ，负荷管理终端，中标比
例 30% ，抄表机，中标比例 100% 。中标公示期为三天。
本次中标金额约为 3,000 万元，占公司 2016 年经审计营业收入的 6.56% 。随着智能电网的快速发展，电力管理对数据的采集和传输手段的要求越来越高，本次中标是对公司采集技术的认可，是公司综合实力的体现。对公司 2018 年经营工作和经营业绩产生积极的影响，但对公司业务、经营的独立性不产生影响。
由于上述项目还未收到正式中标通知书，能否最终中标，并签订正式项目合同，尚存在一定的不确定性。公司将按规定及时披露该项目最终中标结果及后续进展情况，敬请广大投资者注意投资风险。
